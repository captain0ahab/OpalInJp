# OpalInJp — 日本鉱物産地探索マップ

地質データをもとに、日本国内で鉱物が採取できそうなエリアをスコアリングしてマップ上に可視化する Web アプリです。

**[→ デモを見る](https://opalinjp.onrender.com)**（Render 無料枠のため、15分無操作後はスリープします）

---

## 主な機能

- 地図上の任意地点をクリックして調査地点を設定
- 9種類の鉱物から対象を選択（または全鉱物の合計スコア）
- 調査半径を 1〜100 km でスライダー調整
- スコア上位 80 件のポリゴンを鉱物アイコン付きマーカーで表示
  - マーカーのグロー色・強度がスコアに連動（赤→橙→黄→青）
- 結果カードとポップアップに生成プロセス解説を表示
- URL コピーで検索条件をそのままシェア
- スマートフォン対応レスポンシブデザイン
- 国土地理院タイルによる日本語地名表示（ダークモード）

## スコアリングの仕組み

各地質ポリゴンに対して最大 6 項目を評価し、合算したスコアで順位付けします。

### 1. 岩種スコア（Lithology Score）
産総研 地質調査総合センターの **20万分の1地質図** から取得した岩種（`lithology_ja` / `lithology_en`）と、各鉱物の母岩キーワードを照合してスコアを付与します。

| 例：砂金 | キーワード | スコア |
|----------|----------|--------|
| Tier 1 | デイサイト・流紋岩 | 3.0 |
| Tier 2 | 花崗岩・閃緑岩 | 2.0 |
| Tier 3 | 安山岩・玄武岩 | 2.0 |
| Tier 4 | 変成岩 | 1.5 |

### 2. 地質年代ボーナス（Age Bonus）
地層の形成年代が鉱物生成に適した時代（例：砂金→中新世 ×1.5）であれば、岩種スコアに乗数を掛けます。

### 3. 断層近接ボーナス（Fault Bonus）
断層沿いは熱水が循環しやすいため、断層に近接するポリゴンに固定点を加算します（鉱物ごとに設定）。

### 4. 熱水変質スコア（Hydrothermal Score）
- 完新世・後期更新世の火山岩ポリゴンへの距離を評価
  - 0 m（重複）→ +3.0 / 5 km 以内 → +2.0 / 10 km 以内 → +1.0
- OSM の温泉ノードが 5 km 以内にある場合は +1.0 加算（上限 3.0）

### 5. 既知産地スコア（Known Locality Score）
産総研 MRDB・鉱物同好会記録・文献をもとに約 80 箇所の既知産地を登録。ポリゴンと産地の距離に応じてスコアを加算します（信頼度 1〜3 で補正）。

### 6. 河川アクセススコア（River Access Score）
OSM Overpass API を使用して半径 5 km 以内の水系を取得。名前付き河川があれば +3.0、小河川は +2.0。

## ローカルで動かす

### 必要なもの
- Python 3.11+
- `output/mineral_candidates.csv`（解析スクリプトが生成。大容量のため Git 管理外）

### セットアップ
```bash
git clone https://github.com/captain0ahab/OpalInJp.git
cd OpalInJp

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

pip install -r requirements.txt
python app.py
```
ブラウザで `http://localhost:5000` を開きます。

### CSV の再生成（解析スクリプト）
```bash
# 地質図ダウンロード → ポリゴン解析 → 最終 CSV 生成
python analyze_minerals.py
python find_hotspots.py
```
> ⚠️ 解析には数時間〜数十分かかります。大容量ファイルは `.gitignore` で除外しています。

## ライセンスと免責事項

- 本ツールのスコアはあくまでも**地質的な可能性の参考指標**です。実際の採取可否・産出保証ではありません。
- 鉱物採取は**土地所有者・管理者の許可**を必ず得た上で行ってください。国立公園・自然保護区域内での採取は法令で禁止されています。
- 使用する地質データは広域図（20万分の1）のため、局所的な地質変化は反映されていません。
